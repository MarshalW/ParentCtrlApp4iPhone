# 配置和运行项目

本项目使用了CocoaPods。

## 依赖库

使用CocoaPods，安装了如下项目需要的项目：

* AFNetworking  用于HTTP+JSON访问
* OHHTTPStubs   用于针对HTTP的Mock测试


## pod的安装

见[cocoapods](http://cocoapods.org/)

  $ sudo gem install cocoapods


## pod安装项目库

在项目目录下执行：

$ pod install

注意，打开项目，需要打开`xcworkspace`扩展名的文件，别打开`xcodeproj`结尾文件
